# Keyboard Shortcuts

Back to [Docs Index](../README.md)

On macOS, use `Command`.

On Windows and Linux, use `Ctrl`.

## App-Wide

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + N` | New database |
| `Cmd/Ctrl + O` | Open database |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

## Table Zoom

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + Shift + =` | Zoom in |
| `Cmd/Ctrl + Shift + -` | Zoom out |
| `Cmd/Ctrl + Shift + 0` | Reset zoom |

## Transactions View

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + A` | Select all visible rows |
| `Arrow Up / Arrow Down` | Move row focus |
| `Shift + Arrow Up / Arrow Down` | Extend selection |
| `Cmd/Ctrl + Shift + Arrow Up / Arrow Down` | Select from current row to top or bottom |
| `Enter` | Edit focused non-planned transaction |
| `Escape` | Close attachment panel, then clear selection |
| `Delete` | Delete selected rows |
| `A` | Approve selected pending expenses |
| `R` | Reject selected pending expenses |
| `E` | Edit selected row, or bulk edit multiple real rows |

## Planned View

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + N` | Add planned template |
| `Arrow Up / Arrow Down` | Move between templates |
| `Escape` | Clear selected template |
| `E` | Edit selected template |

## Activities View

| Shortcut | Action |
| --- | --- |
| `Arrow Up / Arrow Down` | Move between activities |
| `Enter` | Open selected activity |
| `Escape` | Clear current selection |

## Notes

- Shortcuts that operate on table rows usually ignore you while you are actively typing into an input field.
- Approval shortcuts are aimed at pending expenses, not every transaction type.
- The zoom shortcuts affect table density, not the whole application window.
